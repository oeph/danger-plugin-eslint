> Eslint your code with Danger

_**This fork includes support for Bitbucket Server**_

## Usage

Install:

```sh
yarn add @oeph/danger-plugin-eslint --dev
```

At a glance:

```js
// dangerfile.js
import eslint from '@oeph/danger-plugin-eslint'

eslint()
```
